# ABE Next.js Site

Projeto institucional em Next.js com:
- modo escuro como padrão
- alternância para modo claro
- logos da ABE, AvantPay, Acordo Seguro e Grejo
- seções institucionais completas
- área de contratos baseada no comparativo enviado

## Como rodar

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Publicação no Vercel

1. Envie a pasta para um repositório GitHub.
2. Importe o projeto no Vercel.
3. O framework será identificado automaticamente como Next.js.

## Arquivos de imagem

As logos estão em `public/logos`.
